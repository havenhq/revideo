# Project structure

Revideo Projects generally follow the same folder structure. This is what your project should look like, along with explanations of what important files or folders are responsible for:

```
project/
├── src/
│   ├── scenes/
│   │   └── example.tsx
│   ├── project.meta
│   ├── render.ts
│   └── project.ts
├── public/
│   ├── image-asset.png
│   └── audio-asset.wav
├── package.json
└── tsconfig.json
```

# Player for Preview & Editing

To preview videos in the browser, you can use the Player Component (check it out in action in our [Saas Template](https://github.com/redotvideo/revideo-saas-template/blob/main/ui/src/app/edit/%5Bid%5D/page.tsx)). To use the component, you need to build your revideo project and reference the resulting file in the HTML component. 

To build your project, run the following inside your Revideo project:

```bash
npm run build
```

You will now find a `project.js` file in `/dist`. 

If you want to use this file for the player component, for example in a web app built with React or NextJS, you should copy this file over to your project, for example in the `/public` folder. Now you can display your video in a React / NextJS App:

```tsx
"use client";
import { useEffect, ComponentProps } from "react";
import { MotionCanvasPlayerProps } from "@revideo/player";

declare global {
  namespace JSX {
    interface IntrinsicElements {
      'revideo-player': MotionCanvasPlayerProps & ComponentProps<'div'>;
    }
  }
}

export default function YourComponent() {
  useEffect(() => {
    import("@revideo/player");
  }, []);

  const variables = { fill: "white" }

  return (
      <revideo-player 
        src="/project.js" 
        variables={JSON.stringify(variables)}          
      />
  );
}
```

### Real-Time Editing

When the variables parameter changes, the updated inputs will be reflected instantly in the preview. This enables real-time editing.


### Preloading Assets

If your video uses assets from the internet, you might want to preload assets to avoid a lag in the preview. For example, if you display a sequence of images, with each image being shown for 5 seconds, your browser will only start loading the image once it is supposed to show up on the screen, in this case after 5, 10, 15 seconds and so on.

To avoid this, you can preload images by adding a `<link>` tag and only display the player once all assets are loaded:


```tsx
export default function YourComponent(){
    const [assetsLoaded, setAssetsLoaded] = useState(false);
    const images = ["https://my-image-1.png", "https://my-image-2.png"]

    const variables = { images: images }

    useEffect(() => {
      const preloadLinks: HTMLLinkElement[] = [];
      const totalImages = metadata?.images.length;  
      let loadedImagesCount = 0;
    
      images.forEach((imageUrl: string) => { // preload images
          const link = document.createElement('link');
          link.rel = 'preload';
          link.as = 'image';
          link.href = imageUrl;
          link.crossOrigin = 'anonymous';
          document.head.appendChild(link);
          preloadLinks.push(link);
          const img = new Image();
          img.onload = () => {
            loadedImagesCount++;
            if (loadedImagesCount === totalImages) { // set assetsLoaded = true when all images are loaded
              setAssetsLoaded(true);
            }
          };
          img.src = imageUrl;
      });

      return () => {
        preloadLinks.forEach(link => {
          document.head.removeChild(link);
        });
      };
    }, [metadata]);

    return (
        { assetsLoaded ? ( // display player when assets are loaded, otherwise display "Loading assets..."
            <revideo-player 
                src="/project.js" 
                variables={JSON.stringify(variables)}          
            />
            ) : <div>Loading assets...</div>
        }
    );
}
```
